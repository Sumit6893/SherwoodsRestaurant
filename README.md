# 🍽️ Sherwoods - Multi Cuisine Restaurant Website

A modern, responsive, and premium restaurant website built using **HTML, CSS, and JavaScript**.
This project is designed to provide a smooth and interactive user experience similar to platforms like Swiggy.

---

## 🚀 Live Features

* 🔍 Search functionality (real-time filtering)
* 🥗 Veg / Non-Veg filter system
* 📂 Category-wise menu (Swiggy-style UI)
* 🎨 Premium UI with animations and hover effects
* 📱 Fully responsive (mobile + tablet + laptop)
* 📌 Sticky navbar
* 🖼️ Image-based food cards
* 💬 WhatsApp contact button

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Grid + Animations)
* JavaScript (DOM Manipulation)
